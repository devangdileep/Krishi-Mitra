# Color System

Primary:

- Deep Emerald: `#1F7A4D`
- Soft Green Glow: `#38C172`
- Sky Blue Accent: `#60A5FA`
- Warm Amber: `#FBBF24`

Light Mode:

- Background: `#F5F7F6`
- Glass Surface: `rgba(255,255,255,0.18)`
- Border: `rgba(255,255,255,0.28)`
- Shadow: `rgba(0,0,0,0.08)`

Dark Mode:

- Background: `#0A0F0D`
- Glass Surface: `rgba(255,255,255,0.08)`
- Border: `rgba(255,255,255,0.12)`
- Shadow: `rgba(0,0,0,0.35)`

Avoid flat colors, harsh black, neon green, oversaturation, and heavy borders.
